Urban planning game

Build and improve your county. How large can it get?

Rules

Turn based game where each turn has two actions

  1. Throw a die
  2. Pick a building type to build
  3. Pick a place to build the block of buildings

Once step two is done, the corresponding row of the Contruction Board will grow up to a maximum of six blocks. This determines the size of the block to be built in the County on this turn.

The block of buildings to be constructed should all form a contiguous line and lie in the row or column determined by the die.

Observations

Contiguous plots of land are tiles that share an edge. In this way, we can see that a plot of land in the county can have at most four neighbors.

You cannot place a block with more than six buildings.

All buildings I should be build in continuous plots of land.

You cannot build over polts of land that are already developped, unless you are building I.

End of game

The game ends when you select a building that cannot be build because there is no space in the County for the corresponding block
